CRM Admin Panel

Overview

The CRM Admin Panel is a lightweight web application built with Flask and SQLite. It provides a straightforward interface for managing a list of customers: you can add new entries, edit existing ones and remove obsolete records. The project demonstrates how to build a CRUD app using Python, the Flask microframework and SQLAlchemy.

Key Features

  • List all customers in a sortable table
  • Add new customers with name, email and phone number
  • Edit and delete existing records
  • Persistent storage using SQLite and SQLAlchemy
  • Clean, responsive user interface

How to Run

Navigate to the admin_panel directory and install the dependencies. Then start the development server:

pip install -r ../requirements.txt
python app.py

Open your browser at http://localhost:5000 to access the dashboard.
