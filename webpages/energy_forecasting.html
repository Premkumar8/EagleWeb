Energy Forecasting

Overview

This project demonstrates how to build and train a long short‑term memory (LSTM) neural network to forecast energy consumption. Because no real dataset is bundled with the code, the script synthesises a time series using a sine wave with added noise and then transforms it into a supervised learning problem.

Key Concepts

  • Time‑series generation and preprocessing
  • Transforming sequences into supervised learning datasets
  • Building and training an LSTM with the Keras API
  • Evaluating model performance on a hold‑out set

How to Run

From the energy_forecasting directory, install the dependencies and execute the script:

pip install -r ../requirements.txt
python energy_forecasting.py

The script will print the mean squared error on the test set after training.
