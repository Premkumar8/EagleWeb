Emotion Recognition

Overview

Here we build a simple one‑dimensional convolutional neural network in PyTorch that classifies electrocardiogram (ECG) signals into several emotional states. To keep the example self‑contained, the script creates synthetic signals instead of loading a real biosignal dataset.

Key Concepts

  • Generating synthetic 1‑D signals for classification
  • Implementing a lightweight 1‑D CNN using PyTorch
  • Training with cross‑entropy loss and the Adam optimiser
  • Reporting training loss across epochs

How to Run

From the emotion_recognition directory, install PyTorch and execute:

pip install torch
python emotion_recognition.py

You will see the training loss printed after each epoch.
