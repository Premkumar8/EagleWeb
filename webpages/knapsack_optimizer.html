Knapsack Optimizer

Overview

This example illustrates the classic 0/1 knapsack problem and a dynamic programming solution for maximising value subject to a weight constraint. The algorithm determines which items to put into the knapsack to achieve the highest possible total value.

Key Concepts

  • Dynamic programming and table construction
  • Backtracking to recover the set of chosen items
  • Understanding computational trade‑offs of exhaustive search versus DP

How to Run

From the knapsack_optimizer directory simply run:

python knapsack_optimizer.py

The script prints the maximum value achievable with the given capacity and lists the indices of the selected items.
